- [x] **Task 4: Backend PDF Text Extraction Service** ✅ COMPLETE
  - Date: March 25, 2026
  - Details: `POST /api/extract-text` endpoint in `backend/main.py`
  - Features:
    - Accepts multipart form: `file` (PDF) + `api_key`
